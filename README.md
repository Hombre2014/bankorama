# Bankorama

> Bankorama is a modern web application that allows you to manage your bank accounts.

## Built With

- Major languages: TypeScript
- Backend: Appwrite
- Framework: Next.js
- Tools used: Plaid, Dwolla, Sentry
- Styling: TailwindCSS, shadcn/ui
